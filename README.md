# Refúgio da Preta — Frontend do Integrante 2 integrado à main

Este pacote foi preparado para adicionar a parte de frontend do Refúgio da Preta sem mudar a estrutura principal do backend Flask.

## Estrutura

```text
app.py
database.py
schema.sql
requirements.txt
templates/
static/
```

## Como rodar

```bash
pip install -r requirements.txt
python app.py
```

Abra:

```text
http://127.0.0.1:5000
```
